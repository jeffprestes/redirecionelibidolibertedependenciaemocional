# Redirecione sua Libido – Liberdade da Dependência Emocional

Este projeto é uma Aplicação Web Progressiva (PWA) de página única, dedicada a explorar o tema da dependência emocional e o redirecionamento da energia vital (libido) de forma saudável e autônoma. 

O conteúdo original foi escrito pela psicóloga **Gabriele Garbin**.

## 📖 Sobre o Conteúdo

A dependência emocional ocorre quando a energia vital de uma pessoa depende de objetos externos (pessoas, validação, conquistas) para se ativar. Este artigo explora o processo de integração da libido internamente, onde a energia passa a circular a partir da própria psiquê, resultando em:

- Energia para agir sem precisar de aprovação
- Criatividade natural e autêntica
- Vontade de viver independente de excitação externa constante
- Relacionamentos baseados em verdadeira integração, não em falta ou projeções

## ✨ Características Técnicas

- **Página Única (Single-Page)**: Estrutura semântica HTML5 estática, construída de maneira limpa (sem frameworks complexos).
- **PWA (Progressive Web App)**: Pronta para instalação em dispositivos móveis, com suporte offline via `sw.js` e `manifest.json`.
- **Design Focado e Acolhedor**: UX "trauma-informed" com paleta de cores serenas, foco na legibilidade, espaçamento generoso e animações suaves (com suporte a `prefers-reduced-motion` no CSS).
- **Otimização para Dispositivos Móveis**: Ajustado para o funcionamento correto em "safe-areas" (iPhone 16 / Dynamic Island) e Chrome Android, com tipografia legível e fácil interação.
- **SEO Otimizado**: Tags Open Graph, Twitter Cards e JSON-LD (`Article`) para indexação completa pelas ferramentas de busca (Google).
